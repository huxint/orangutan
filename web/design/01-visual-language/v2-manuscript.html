Body · Newsreader Regular

Orangutan 是一个 C++23 的 agent 助手，前端的任务是把它庞杂的 ReAct 循环变成一份可被轻松阅读的文档。手稿方案用的是单家族字族策略（所有字都来自 Newsreader），靠字重、斜体和字号做层级，而不是靠第二种字体——这样排版噪声最低。

Spline Sans Mono 做等宽，它的 tabular 与 Newsreader 的正文垂直节律意外地吻合，又比 IBM Plex Mono 柔和，像学者自己手写在 code 缝隙里的笔记。

“单家族的好处不是视觉统一，而是语气统一 —— 每一行字都像出自同一个人之手。
